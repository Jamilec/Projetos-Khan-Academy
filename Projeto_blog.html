Aprendendo o que é o VS Code

Postado em 02/11/20

Meu primeiro passo no mundo da programação foi baixar esse programa e aprender a usá-lo, o legal é que ele bem simples e intuitivo. O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.

Aprendendo que é o Git

Postado em 02/11/20

Depois de entender um pouco sobre o editor de códigos, foi necessário aprender o que é o Git. Nessa parte eu tive um pouco de dificuldade, mas o legal é que o git é repetitivo, então, o mais importante é aprender o básico do Git. O Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.
